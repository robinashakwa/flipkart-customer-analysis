# Flipkart Customer Sentiment & Product Analysis

## 🚀 Project Overview

This project analyzes Flipkart customer reviews to understand how pricing and product quality influence customer sentiment.

It demonstrates a complete data analysis workflow using Python, SQL, and Excel.

---

## 🧠 Problem Statement

How do product pricing and customer ratings impact customer sentiment?

---

## ⚙️ Workflow

1. Data Cleaning using Python
2. Feature Engineering (price range, rating category)
3. Data Storage using SQL (SQLite)
4. Data Analysis using SQL queries
5. Automated Excel report generation

---

## 🛠️ Tech Stack

* Python (Pandas)
* SQL (SQLite)
* Excel (Reporting)
* VS Code

---

## 📊 Key Insights

* Majority of reviews are positive
* Medium-priced products dominate
* Ratings strongly influence sentiment

---

## 📁 Files Included

* `pipeline.py` → Full automation script
* `Raw_Dataset.csv` → Dataset
* `flipkart_report.xlsx` → Final report

---

## 🚀 How to Run

1. Open `pipeline.py`
2. Run the script
3. Excel report will be generated automatically

---

## 🎯 Outcome

Built a complete end-to-end data analysis pipeline combining Python, SQL, and Excel.
